# Reto Integrador Unidad 1 - TechVentas S.A.S

## Descripción
Se realiza un analisis de datos de ventas del primer semestre de la compañia TechVentas S.A.S con el objetivo de realizar un pipeline completo de este semestre.
Pipeline completo: álgebra lineal con Numpy, exploración de datos con SQLite y transformacíon con Pandas

## Estructura del proyecto

TECHVENTAS S.A.S
├── data/
├── notebooks/
├── sql/
├── output/
├── setup.md
├── README.md
└── requirements.txt

## Herramientas usadas
- Python 3.11
- Numpy
- Pandas
- SQLite3

## Cómo ejecutarlo

1. Sigue los pasos del archivo `setup.md`
2. Abre Jupyter Notebook
3. Ejecuta los notebooks en orden
