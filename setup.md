# Setup del proyecto - TechVentas S.A.S

## Requisitos
- Python 3.11
- Anaconda

## 1. Crear el entorno virtual
"""bash
conda create -n reto_u1 python=3.11
conda activate reto_u1
"""

## 2. Instalar dependencias
"""bash
pip install numpy pandas
"""

## 3. Clonar el repositorio y ubicarse en la carpeta
"""bash
cd ruta/del/proyecto
"""

## 4. Ejecutar los notebooks en orden
1. `notebooks/01_numpy.ipynb`
2. `notebooks/02_sql_discovery.ipynb`
3. `notebooks/03_pandas_pipeline.ipynb`
